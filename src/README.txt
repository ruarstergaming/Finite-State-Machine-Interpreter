to run fsmintepreter:
  1 - compile the 2 .java files
  2 - run java fsmintepreter <fsm file path> 
  or if you have a txt file of test inputs
  3 - run java fsmintepreter <fsm file path> <test input file path>
  
the main method is stored in fsmintepreter class and will accept either a single file name or 2 fiel names as arguments
